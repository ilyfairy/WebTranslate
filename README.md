# WebTranslate

使用的WebView2  

![WebTranslate](https://github.com/ilyfairy/WebTranslate/blob/master/Img/WebTranslate.webp)

## 介绍  

现在支持以下翻译  
- 谷歌翻译 translate.google.cn
- 百度翻译 fanyi.baidu.com

去除了一些广告以及多余的内容  

## 配置  

Ctrl+Shift+T  打开翻译  
Ctrl+Tab|Shift  切换翻译  
Ctrl+Shift+S  交换语言  
Ctrl+F12  打开设置
